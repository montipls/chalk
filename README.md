# 2D physics engine
A humble physics engine made to be used in other projects.

### How to run
1. Install make & gcc
2. Clone this repository: `git clone https://github.com/montipls/chalk.git`
3. Navigate to the folder: `cd chalk`
4. Compile & run: `make run`
